Class: LoadingManager

LoadingManager

new LoadingManager(onLoad, onProgress, onError)

Class representing an observer for Loaders. It's instance may be used to monitor multiple loaders.
Parameters:
Name Type Description
onLoad function Will be called when the Loader X notifies that the item I finished loading
onProgress function Will be called when the Loader X sends progress notification during the loading of item I.
onError function Will be called when the Loader X encounters an error during the loading.
