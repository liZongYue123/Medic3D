parse(text) → {Array}

Parses the received text and returns the array of objects. Each object has geometry, material and name property. The geometry property holds the arrays of normals, uvs and vertices. The material property holds the name of the object material and smooth shading flag.
Parameters:
Name Type Description
text string Text in Wavefront OBJ geometry format.
